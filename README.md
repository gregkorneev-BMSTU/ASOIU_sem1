# 🏁 [Seminar 1 — Анализ квалификации Гран-При](https://github.com/ugapanyuk/course-architecture/blob/main/seminars/seminar1.pdf)

## 📚 Описание проекта
Проект реализует анализ квалификационных результатов команд (Гран-При) в рамках дисциплины АСОИУ.

Программа:
- принимает данные о командах и их средней скорости
- вычисляет статистику
- сортирует результаты
- выполняет фильтрацию

Проект содержит **две версии**:
- базовую (по методичке)
- улучшенную (с расширениями)

# ▶️ Запуск проекта

## 📌 Версия 1 (Seminar1)

```bash
cd Seminar1
dotnet run
```

📌 Особенности:
- ввод данных с клавиатуры
- реализация строго по методичке

---

## 📌 Версия 2 (Seminar1_ver2-upd)

```bash
cd Seminar1_ver2-upd
dotnet run
```

При запуске:

1 — ввод с клавиатуры
2 — чтение из файла

📌 Если выбран режим 2, используется файл:
```
input.txt
```

# 🥇 Версия 1 — Базовая

## Особенности
- массивы
- циклы for
- функции
- Console + Parse

## Ограничения
- нет проверки ввода
- возможны ошибки
- зависимость от формата чисел

---

# 🚀 Версия 2 — Улучшенная

## Улучшения

### Безопасный ввод
- TryParse
- защита от пустых значений

### Работа с числами
- поддержка "." и ","

### Работа с файлом
- чтение input.txt
- автоматический путь

### Валидация
- проверка структуры файла
- проверка данных
